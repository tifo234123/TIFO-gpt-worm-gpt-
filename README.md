# TIFO-GPT / WormGPT Clone

> An AI assistant project built for **educational and research purposes**, focusing on prompt engineering, tool integration, and conversational workflows.

---

## 📦 Installation
```bash
git clone https://github.com/tifo234123/TIFO-gpt-worm-gpt-
cd TIFO-gpt-worm-gpt-
```

### Install Dependencies

**Linux:**
```bash
python3 -m pip install requests
```

**Windows:**
```cmd
py -m pip install requests
```

---

## 🔑 API Key Setup

Get your API key from [OpenRouter](https://openrouter.ai/) and export it:

**Linux:**
```bash
export OPENROUTER_API_KEY=your_key_here
```

**Windows (CMD):**
```cmd
set OPENROUTER_API_KEY=your_key_here
```

---

## 🚀 Usage

**Linux:**
```bash
python3 main.py
```

**Windows:**
```cmd
py main.py
```
